GapAnalysisResult CareerNavigator.Core.Engines.GapAnalyzer.AnalyzeGap ( AnalysisResult user,
AnalysisResult job )

Performs a comprehensive gap analysis between a user profile and a target job.

The analysis involves three phases:

  1. Direct Skills Gap: Identifies skills explicitly required by the job but missing from the user.
  2. Implicit Dependency mapping: Uses the Universe graph to find related skills (neighbors) that might bridge the gap.
  3. Seniority/Authority Scaling: Compares the structural seniority (Senior, Managerial) of the roles.
Parameters
userThe user's analyzed profile.
jobThe job's analyzed requirements.
Returns
A detailed GapAnalysisResult highlighting missing skills and seniority mismatches.
