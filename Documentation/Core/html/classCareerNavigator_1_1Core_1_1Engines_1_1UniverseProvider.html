Detailed Description

Singleton service responsible for managing the in-memory graph Universe.

  • Loads the universe.json file generated by the Python DataFactory.
  • Builds optimized lookup structures (Dictionary<TKey,TValue> indices) for O(1) performance.
  • Watches for file changes to support hot-reloading of data without restart.
