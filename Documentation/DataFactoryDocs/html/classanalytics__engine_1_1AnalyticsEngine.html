Detailed Description

Pure functional engine for calculating derived statistical metadata.

Unlike GraphBuilder, which manages state, this class performs stateless mathematical transformations on the data to produce insights (e.g., histograms, percentiles).

Computes a histogram of skill seniority scores.

Buckets skills into ten distinct bins ranging from 0.0 to 1.0 (e.g., "0.0-0.1", "0.9-1.0"). This is used by the frontend to visualize the "Seniority Spread" of the entire technology universe.

Parameters
seniority_scoresA list of float scores (0.0 to 1.0) derived from skill analysis.
total_skillsThe total count of skills analyzed.
Returns
A dictionary containing the histogram bucket map and total count.
